# 情绪识别系统

## 功能简介

来福可以通过摄像头"偷偷观察"主人，识别你的表情和情绪，并做出自然的反应。

- **开心时**：来福跟着摇尾巴
- **难过时**：来福会蹭过来安慰你
- **生气时**：来福缩起身子，小心翼翼地看着你
- **惊讶时**：来福好奇地歪头
- **疲惫/持续低落时**：来福会主动搭话关心你

来福观察时会有微妙的小动作（竖耳朵、歪头），像真正的狗狗偷偷看你一样。

---

## 配置步骤

### 1. 安装 DeepFace 依赖

情绪识别需要 DeepFace 库。在 `python-service/` 目录下运行：

```bash
pip install deepface tf-keras
```

> DeepFace 首次运行时会自动下载人脸检测模型（约 500MB），需要联网。后续使用无需联网。

### 2. 配置视觉 LLM（深度分析，可选）

深度分析功能需要支持图像输入的 LLM。在 `python-service/config.json` 的 `llm` 部分添加 `vision_model` 字段：

```json
{
  "llm": {
    "provider": "openai",
    "base_url": "https://api.deepseek.com/v1",
    "api_key": "你的API密钥",
    "model": "deepseek-chat",
    "vision_model": "deepseek-chat"
  }
}
```

| 字段 | 说明 |
|------|------|
| `vision_model` | 支持图像输入的模型名称。如果你的 LLM 提供商支持多模态（如 GPT-4o、Gemini），填写对应模型名。如不配置，默认使用 `model` 字段的值 |

> **不配置视觉 LLM 也能用**：本地表情检测（Layer 1）完全离线运行，来福仍然会根据检测到的表情做动作反应。深度分析（Layer 2）只在情绪明显变化时才触发，用于让来福主动搭话关心你。

### 3. 在应用中启用

1. 打开设置面板（系统托盘右键 → 设置）
2. 找到 **👁️ 情绪观察** 区域
3. 打开「让来福观察你」开关
4. 首次开启时，来福会好奇地问你：*"汪？主人，来福想看看你的样子！可以让来福偷偷看你吗？"*
5. 点击「好呀！」→ 浏览器弹出摄像头权限弹窗 → 允许
6. 完成！来福开始偷偷观察你

## 设置选项

| 选项 | 默认值 | 范围 | 说明 |
|------|--------|------|------|
| 让来福观察你 | 关闭 | 开/关 | 主开关，控制整个情绪识别功能 |
| 偷看频率 | 每 10 分钟 | 5-30 分钟 | 来福自动偷看的间隔时间 |
| 让来福看看你（按钮） | — | — | 手动触发一次完整检测（含深度分析） |

## 工作原理

### 两层检测

| 层级 | 方式 | 速度 | 联网 | 触发条件 |
|------|------|------|------|----------|
| **Layer 1** 本地快速检测 | DeepFace 表情分类 | < 200ms | 不需要 | 每次触发都运行 |
| **Layer 2** LLM 深度分析 | 视觉 LLM 图像理解 | 2-5s | 需要 | 情绪明显变化或持续低落时 |

### 三种触发方式

1. **定时轮询**：按设置的频率自动偷看（默认每 10 分钟）
2. **事件驱动**：在你发送消息前、来福闲置超过 5 分钟、你超过 20 分钟没互动时自动触发
3. **手动触发**：在设置面板点击「让来福看看你」按钮

### 深度分析升级条件

Layer 1 检测完成后，满足以下任一条件会自动升级到 Layer 2：

- 情绪类别发生跳变（如从开心变成难过）
- 连续 2 次检测到负面情绪（sad/angry/fear）
- 手动触发时强制执行
- 两次深度分析之间至少间隔 **3 分钟**

### 摄像头使用策略

- 每次截帧时打开摄像头 → 拍一帧 → **立即关闭**
- 不保持摄像头常开，不影响其他应用使用摄像头
- 如果连续 3 次未检测到人脸（你离开了座位），来福会自动降低偷看频率

## 隐私说明

| 数据类型 | 处理方式 |
|----------|----------|
| 摄像头截帧 | 检测完成后 **立即销毁**，不保存到磁盘 |
| 情绪标签 | 仅保存分类结果（如"happy"）和置信度，存储在内存中 |
| LLM 文字描述 | 深度分析的文字描述保存在内存中，用于对话上下文 |
| 人脸特征向量 | **绝不保存** |
| 原始图像 | **绝不保存** |

> **深度分析时**：截帧图像会通过 API 发送给你配置的视觉 LLM 服务提供商，遵循该提供商的数据政策。如果你不希望图像发送给外部服务，可以只使用 Layer 1（不配置 `vision_model`），来福仍然会根据本地检测结果做动作反应。

## 来福的反应

### Layer 1 快速反应（动作）

检测到表情后，来福立即做出对应动作：

| 你的表情 | 来福的反应 |
|----------|----------|
| 😊 开心 | 轻摇尾巴，回到正常待机 |
| 😢 难过 | 慢慢走近，低头蹭你 |
| 😠 生气 | 缩一下身子，小心翼翼地看着你 |
| 😲 惊讶 | 耳朵竖起，歪头好奇 |
| 😐 平静 | 小呵欠，继续待机 |
| 😨 紧张 | 贴近你，做保护姿态 |

### Layer 2 深度反应（主动搭话）

当深度分析触发时，来福会根据场景主动说话：

- 检测到你趴在桌上 → *"主人...要不要休息一下？来福给你暖脚！"*
- 检测到你看起来很累 → *"主人工作辛苦了，来福陪你！"*
- 检测到你在笑 → *"汪！主人好开心，来福也开心！"*

来福说话时不会用机械的语气（如"我检测到你很难过"），而是像一只敏感的狗狗那样用行动和关心来回应。

## 常见问题

### Q: 开启后摄像头灯一直亮？

不会。来福每次只在截帧瞬间打开摄像头（不到 1 秒），拍完立即关闭。你不会看到摄像头指示灯持续亮起。

### Q: 没有配置视觉 LLM，能用吗？

可以。Layer 1 完全本地运行，来福会根据检测到的表情做动作反应。只是不会有主动搭话功能。

### Q: 首次使用时模型下载很慢？

DeepFace 首次运行需要下载人脸检测模型（约 500MB）。如果下载慢，可以：
- 设置 pip 镜像源加速
- 或手动下载模型文件放到 `~/.deepface/weights/` 目录

### Q: 来福一直检测不到人脸？

检查以下几点：
- 摄像头是否被其他应用占用
- 光线是否太暗
- 人脸是否正对摄像头（侧脸识别率较低）
- 连续 3 次检测不到人脸后，来福会自动降低偷看频率，检测到人脸后恢复正常

### Q: 如何关闭这个功能？

在设置面板中关闭「让来福观察你」开关即可。来福会立即停止所有观察行为。

## API 参考

### POST /api/emotion

手动调用情绪检测接口（用于调试）。

**请求** (`multipart/form-data`)：

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `image` | File | 是 | JPEG 截帧图片 |
| `deep` | bool | 否 | 是否强制深度分析，默认 `false` |

**响应**：

```json
{
  "has_face": true,
  "local": {
    "emotion": "happy",
    "confidence": 0.85,
    "all_scores": {
      "happy": 0.85, "sad": 0.02, "angry": 0.01,
      "surprise": 0.05, "neutral": 0.05, "fear": 0.01, "disgust": 0.01
    }
  },
  "deep": {
    "description": "主人在微笑，看起来心情不错",
    "suggested_action": "celebrate",
    "suggested_speech": "汪！主人好开心，来福也开心！"
  },
  "changed": true
}
```

**curl 测试**：

```bash
curl -X POST http://127.0.0.1:18765/api/emotion \
  -F "image=@photo.jpg" \
  -F "deep=true"
```
