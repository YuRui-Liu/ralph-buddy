# ralph-buddy 部署指南

## 快速开始

### 1. 克隆项目

```bash
git clone <repo-url>
cd pet_buddy
```

### 2. 配置

```bash
# 复制配置模板
cp python-service/config.example.json python-service/config.json
cp python-service/.env.example python-service/.env

# 编辑 .env，填入 API key（必填 LLM_API_KEY）
```

### 3. 安装依赖

```bash
# Python 依赖
cd python-service
pip install -r requirements.txt

# Node 依赖
cd ..
npm install
```

### 4. 启动

```bash
# 开发模式
npm run dev

# 生产模式
npm run build
npm start
```

## 配置方式

### 方式一：编辑 .env（推荐）

```bash
LLM_API_KEY=sk-your-key-here
WHISPER_MODEL_PATH=/path/to/faster-whisper-small
```

### 方式二：编辑 config.json

直接修改 python-service/config.json 中的对应字段。注意不要将 API key 写入此文件。

### 方式三：环境变量（Docker/CI）

```bash
export LLM_API_KEY=sk-your-key-here
export PYTHON_PORT=8000
```

## 本地模型（可选）

### Whisper STT 模型

不配置路径时自动从 HuggingFace 下载（首次需联网，约 244MB）。

手动安装：下载 faster-whisper-small 模型放到任意目录，在 .env 中设置 `WHISPER_MODEL_PATH=/path/to/model`

### Embedding 模型

不配置时使用 Chroma 默认 embedding（首次自动下载约 80MB）。

### GPT-SoVITS 声音克隆

不配置时跳过，使用 Edge TTS 在线语音。如需声音克隆，安装 GPT-SoVITS 到任意目录，设置 `GPT_SOVITS_DIR`。

## 常见问题

### Q: 启动后提示 config.json 不存在？
复制 `config.example.json` 为 `config.json` 即可。不复制也能启动（使用默认值），但需要在 .env 中设置 LLM_API_KEY。

### Q: 如何更换 LLM 提供商？
修改 config.json 中的 `llm.base_url` 和 `llm.model`。只要兼容 OpenAI API 格式即可。

### Q: 端口被占用？
在 .env 中设置 `PYTHON_PORT=其他端口号`。
