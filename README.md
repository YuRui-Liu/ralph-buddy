# ralph-buddy

> 一只有灵魂的桌面 AI 宠物。

Ralph Buddy（来福）是一款桌面 AI 陪伴宠物，基于 Electron + Vue 3 + Python FastAPI 构建。它不只是一段动画——来福拥有长期记忆、情绪感知、性格成长和自主行为，会随着你们的相处逐渐变成独一无二的伙伴。

---

## 核心特性

### 对话与语音
- **自然对话**：接入 LLM（DeepSeek / GPT 等），来福用狗狗的思维方式和你聊天
- **语音识别**：Whisper 本地离线 STT，支持 VAD 自动检测说话
- **语音合成**：Edge TTS 在线语音 + GPT-SoVITS 声音克隆，来福可以用你定制的声音说话
- **智能路由**：预合成片段 → 缓存 → 克隆推理 → Edge TTS 四级回退

### 记忆系统
- **三层记忆**：短期对话（内存）→ 语义检索（向量数据库）→ 长期存储（SQLite）
- **用户画像**：自动从对话中提取你的信息，来福会记住你的喜好
- **记忆压缩**：对话满 10 轮自动摘要压缩，不丢失重要信息

### 情绪识别
- **摄像头观察**：来福通过摄像头"偷偷看"你，识别你的表情和情绪
- **混合检测**：本地 DeepFace 快速识别 + 视觉 LLM 深度理解
- **分层反应**：快速动作反应（摇尾巴/蹭你）+ 主动搭话关心

### 做梦系统
- **睡眠做梦**：来福睡着后会做梦，梦境由 LLM 生成
- **梦境绘图**：硅基流动 AI 生成梦境图片
- **梦境日记**：所有梦境记录可浏览，云朵气泡展示

### 行为与性格
- **12+ 行为脚本**：学究模式、侦探模式、谄媚、舔屏、撒尿、睡前仪式等
- **天性模式**：来福自动做随机行为，像真正的宠物有自己的生活
- **性格成长**：对话会逐渐塑造来福的性格——毒舌值、顺从度、亲密度随互动漂移
- **6 项属性**：健康、心情、精力、亲密度、顺从度、毒舌值，影响说话风格和行为

### 互动方式
- **语音对话**：直接说话，VAD 自动检测
- **文字输入**：键盘打字
- **喂食 / 玩耍**：底部按钮，恢复健康和精力
- **拖拽**：鼠标拖动来福
- **休息提醒**：工作 45 分钟后来福提醒你休息

## 快速开始

### 1. 配置

```bash
# 复制配置模板
cp python-service/config.example.json python-service/config.json
cp python-service/.env.example python-service/.env

# 编辑 .env，填入 LLM API key（必填）
```

### 2. 安装依赖

```bash
# Python
cd python-service
pip install -r requirements.txt

# Node
cd ..
npm install
```

### 3. 启动

```bash
# 开发模式
npm run dev

# 生产模式
npm run build && npm start
```

详细配置说明见 [配置文档](docs/发布部署/配置说明.md) | [部署指南](docs/发布部署/部署指南.md)

## 配置

所有配置通过 `config.json` + `.env` 管理，支持环境变量覆盖：

| 配置 | 必填 | 说明 |
|------|------|------|
| `LLM_API_KEY` | **是** | LLM 服务密钥（DeepSeek/OpenAI 等） |
| `IMAGE_API_KEY` | 否 | 梦境绘图密钥（硅基流动），不填跳过图片生成 |
| `WHISPER_MODEL_PATH` | 否 | Whisper 模型路径，空则自动下载 |
| `PYTHON_PORT` | 否 | 服务端口，默认 18765 |

完整配置项见 [配置说明](docs/发布部署/配置说明.md)
